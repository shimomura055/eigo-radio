# 🌊「20％の料金所は一夜で撤去。それでも、原油市場は静まらない」

ホルムズ海峡に突然、巨大な「料金所」の看板が立った。

2026年7月13日、トランプ米大統領は、海峡を通る全貨物から20％を「安全確保の対価」として受け取る考えを表明。ところが翌14日には撤回し、湾岸諸国による米国への貿易・投資案件へ置き換えると発表した。金額も参加国も、まだ見えない。

「海を通るなら2割払え」

から、

「その代わり、米国に投資を」

へ。

世界のエネルギー輸送を揺らした構想は、わずか1日で姿を変えた。

では、市場は胸をなで下ろしたのか。

答えは、**半分だけ**だ。

ブレント原油は、米国とイランの対立激化や20％案を受け、前日に約10％急騰。撤回後は一時87ドル超から上げ幅を縮めたが、14日の終値は84.73ドルと約2％高、1カ月ぶりの高値だった。通航料は消えても、イラン関係船への封鎖と軍事攻撃は残ったからだ。市場が見ていたのは「料金表」より「海峡の安全」だった。
